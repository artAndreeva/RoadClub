# Road Club

Адаптивная верстка типового одностраничного сайта, который знакомит посетителя с обзором мест для путешествия по стране. Сайт адаптирован под популярные разрешения экранов:  320, 768, 1024 и 1280.

## Технологии

* Flexbox
* Grid
* Git
* Bash
* BEM
* Nested BEM
